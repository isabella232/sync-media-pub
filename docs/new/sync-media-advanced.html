Advanced SyncMedia

Description

Expand what's possible in SyncMedia by defining additional media object types as well as properties.

Use cases

Model

Additional media objects:

  • ref
  • video
  • image

Additional media object properties:

  • panZoom
  • sync:clipPath

Not included

Just so I remember - here are suggestions that weren't included.

  • sync:media: this would be an authoring/packaging convenience but is not tied to end-user benefits.
  • sync:mediaref: without sync:media, we don't need this
  • sync:selector: fragment selectors are in the src property.
